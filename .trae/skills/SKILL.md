# code-uniapp-vue2 技能

## 技能概述

**code-uniapp-vue2** 是一个专注于 uniapp Vue2 项目开发的全能技能，基于 JavaScript 和 Vue 2 官方文档的核心概念，提供代码分析、性能优化和功能实现的综合解决方案。

## JavaScript 核心概念

### 1. JavaScript 基础

-   **变量声明**：`var`、`let`、`const` 的区别和最佳实践
-   **数据类型**：基本类型（String、Number、Boolean、Null、Undefined、Symbol、BigInt）
-   **运算符**：算术、比较、逻辑、赋值、位运算符等
-   **控制流**：`if-else`、`switch`、三元运算符

### 2. 函数和对象

-   **函数定义**：函数声明、函数表达式、箭头函数
-   **参数处理**：默认参数、剩余参数、展开运算符
-   **对象操作**：属性访问、方法调用、解构赋值
-   **类和继承**：ES6 class 语法、原型链继承

### 3. 数据结构

-   **数组操作**：遍历、映射、过滤、归约等方法
-   **对象方法**：`Object.keys()`、`Object.values()`、`Object.entries()`
-   **Map 和 Set**：键值对集合和唯一值集合
-   **WeakMap 和 WeakSet**：弱引用集合

### 4. 异步编程

-   **回调函数**：处理异步操作的传统方式
-   **Promise**：异步操作的对象表示
-   **async/await**：异步编程的同步写法
-   **事件循环**：JavaScript 的运行时模型

### 5. DOM 操作

-   **元素选择**：`getElementById`、`querySelector`、`querySelectorAll`
-   **属性操作**：`getAttribute`、`setAttribute`、`removeAttribute`
-   **样式操作**：`style` 属性、`classList` API
-   **事件处理**：`addEventListener`、事件委托

### 6. 错误处理

-   **try-catch-finally**：异常捕获和处理
-   **throw 语句**：自定义异常抛出
-   **错误类型**：`Error`、`TypeError`、`SyntaxError` 等

### 7. ES6+ 特性

-   **解构赋值**：数组和对象的解构
-   **模板字符串**：多行字符串和变量插值
-   **扩展运算符**：数组和对象的展开
-   **箭头函数**：简洁的函数语法
-   **Promise**：异步操作处理
-   **async/await**：异步编程的同步写法

### 8. 内存管理

-   **垃圾回收**：JavaScript 的自动内存管理
-   **内存泄漏**：常见原因和避免方法
-   **WeakMap 和 WeakSet**：用于避免内存泄漏

### 9. 客户端 Web API

-   **DOM API**：文档对象模型操作
-   **BOM API**：浏览器对象模型操作
-   **Fetch API**：网络请求
-   **LocalStorage 和 SessionStorage**：本地存储

### 10. 模块化

-   **ES Modules**：`import` 和 `export` 语法
-   **CommonJS**：Node.js 模块系统
-   **模块加载器**：处理模块依赖

## Vue 2 核心概念

### 1. 渐进式框架

-   Vue 被设计为可以自底向上逐层应用
-   核心库只关注视图层，易于上手
-   可与第三方库或既有项目整合
-   支持构建复杂的单页应用

### 2. 声明式渲染

-   使用简洁的模板语法声明式地将数据渲染进 DOM
-   数据和 DOM 建立关联，实现响应式更新
-   使用 `v-bind` 指令绑定元素属性
-   支持文本插值 `{{ message }}`

### 3. 条件与循环

-   使用 `v-if`、`v-else`、`v-else-if` 进行条件渲染
-   使用 `v-for` 指令渲染列表
-   支持数组和对象的迭代

### 4. 表单输入绑定

-   使用 `v-model` 实现表单元素和数据的双向绑定
-   支持文本、复选框、单选按钮等表单元素
-   支持修饰符如 `.lazy`、`.number`、`.trim`

### 5. 组件化应用构建

-   将应用拆分为独立可复用的组件
-   每个组件包含自己的模板、脚本和样式
-   支持组件间通信（props、自定义事件、插槽）

### 6. 生命周期钩子

-   `beforeCreate`、`created`、`beforeMount`、`mounted`
-   `beforeUpdate`、`updated`、`beforeDestroy`、`destroyed`
-   用于在组件生命周期的不同阶段执行代码

### 7. 计算属性和侦听器

-   计算属性：基于响应式数据进行计算，具有缓存机制
-   侦听器：监听数据变化并执行副作用

### 8. 自定义事件

-   组件通过 `$emit` 触发自定义事件
-   父组件通过 `v-on` 监听子组件事件
-   支持事件修饰符

### 9. 插槽

-   用于向组件内部传递内容
-   支持默认插槽、具名插槽和作用域插槽

### 10. 混入

-   可复用的组件选项
-   支持全局混入和局部混入

### 11. 插件

-   扩展 Vue 功能的插件系统
-   可添加全局方法、全局资源等

### 12. 单文件组件

-   `.vue` 文件包含模板、脚本和样式
-   支持 CSS 预处理器和模块

## 最佳实践

### JavaScript 最佳实践

-   使用 `const` 优先，`let` 次之，避免使用 `var`
-   遵循严格相等 `===` 而非宽松相等 `==`
-   使用箭头函数简化函数定义，注意 `this` 绑定
-   合理使用解构赋值简化代码
-   使用模板字符串替代字符串拼接
-   异步操作使用 Promise 或 async/await
-   避免回调地狱，使用 Promise 链或 async/await
-   合理使用 `try-catch` 处理异常
-   使用 `let` 和 `const` 的块级作用域
-   遵循 ES6+ 语法规范

### Vue 2 最佳实践

-   遵循 Vue 2 官方风格指南
-   使用组件化开发思想
-   合理使用计算属性和侦听器
-   避免不必要的重渲染
-   使用 v-if 和 v-for 时注意性能
-   合理使用生命周期钩子
-   遵循单向数据流原则
-   组件命名使用 PascalCase 或 kebab-case
-   使用 `key` 属性优化列表渲染
-   避免在模板中使用复杂表达式
